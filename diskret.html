Diskret matematik

The course introduces some of the basic elements of discrete mathematics, set theory, combinatorics, modular arithmetic, propositional and predicate logic and graph theory.

The course includes the following elements:

  • Basic integer math, modular arithmetic
  • Numbers in different number bases
  • Quantitative learning with applications
  • Combinatorics with applications
  • Mathematical recursion and induction
  • RSA encryption and other cryptos
  • Functions and relations between finite quantities, equivalence relations
  • Basic graph theory
  • Propositional logic, predicate logic, logical inference.
